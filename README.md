# A simple showcase of image based color theme

## Setup

> Python > 3.6

> Node Js > 10.0.0

Python Dependencies

```shell
pip install Pillow colorthief flask
```

Npm setup
```shell
npm i
```

### Note

To change `:root` vars in css through js 

```javascript
document.documentElement.style.setProperty('--your-variable', '#YOURCOLOR');
```
